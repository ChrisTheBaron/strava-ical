# strava-ical

**Currently proof of concept.**

Uses a lot of the same code as [ury-ical](https://github.com/UniversityRadioYork/ury-ical).

## Running

```bash
$ go build
$ ./strava-ical -c config.toml
```

## Endpoints

## `/strava.ics`

All activities for the athlete.
